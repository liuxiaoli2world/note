# 本地和远程合并

> 背景：本地的文件，推送到远程。
>
> 1. 建立远程库
> 2. 本地下载
> 3. 本地推送到远程库

```shell
git add .
git commit -m 'xxxx'
# https://github.com/liuxiaoli2world/myPythonCode.git是建立的远程库地址
git remote add origin https://github.com/liuxiaoli2world/myPythonCode.git
# 本地下载
git pull origin master --allow-unrelated-histories
# 推送到远程
git push origin master
```
